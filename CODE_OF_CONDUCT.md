# Code of Conduct

The Lonca community adopts version 2.1 of the **Contributor Covenant** as its code of conduct.

The full text is the canonical source of truth:

📜 [Contributor Covenant 2.1 — English](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)

## In short

We are committed to making participation in this project a welcoming and respectful experience for everyone, regardless of background or identity.

Be kind, assume good intent, give constructive feedback, and focus on what is best for the community.

Behavior that does not align with this standard will not be tolerated.

## Scope

This Code of Conduct applies to all project spaces — the GitHub repository (issues, PRs, Discussions, commits), any official communication channels, and public events where someone represents the project.

## Reporting

If you experience or witness conduct that violates this code, please contact the maintainers privately:

- Email: **conduct@lonca.dev** (will be active once the domain is provisioned)
- Or via a [GitHub private security advisory](https://github.com/loncadev/lonca/security/advisories/new) (use the "Other" category for non-security reports)

All reports will be reviewed and investigated promptly and fairly. The maintainers are obligated to respect the privacy and safety of the reporter.
