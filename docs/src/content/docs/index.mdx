## Why Lonca?

<CardGrid>
  <Card title="Live-verified" icon="approve-check">
    Every endpoint typed from a real SIT sandbox response, not just an OpenAPI spec.
  </Card>

<Card title="Spec-faithful" icon="document">
  Every method signature mirrors what Hepsiburada / Trendyol publish — including quirks like
  per-host path casing.
</Card>

<Card title="No magic" icon="setting">
  Plain `fetch`, plain TS. No code generation; no opaque proxies. Read the source.
</Card>
